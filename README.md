# 🧮 Calculadora em C

Este projeto é uma calculadora simples desenvolvida em linguagem C, capaz de realizar diversas operações matemáticas entre dois números inteiros.

## 🚀 Funcionalidades

O programa permite realizar:

1. Soma
2. Subtração (a - b)
3. Subtração (b - a)
4. Multiplicação
5. Divisão (a / b)
6. Divisão (b / a)
7. Quociente inteiro (a / b)
8. Quociente inteiro (b / a)
9. Resto da divisão (a % b)
10. Resto da divisão (b % a)

## ⚠️ Tratamento de erros

* Verifica divisão por zero
* Valida se a operação escolhida é válida

## 💻 Como executar

1. Compile o código:

```
gcc calculadora.c -o calculadora
```

2. Execute o programa:

```
./calculadora
```

3. Insira:

* Dois números inteiros
* O número da operação desejada

## 🧠 Exemplo de uso

Entrada:

```
10 5 1
```

Saída:

```
15
```

## 📚 Tecnologias utilizadas

* Linguagem C
* Compilador GCC
